How to use this page (simple)

  1. Fill the four boxes below (topic, keywords, links if any, raw ideas).
  2. Click Generate blog post. Your drafts appear in the Blog post and LinkedIn tabs right under the button.
  3. If that button errors, open Plan B (ChatGPT) and follow the two steps there instead.
  4. To put the article on your site: check date and web address (slug), then either Download .md and add the file to Git yourself, or Publish to GitHub if you saved a token in Keys at the bottom.
