# Shabeer Mohamed - Portfolio Website

A bold, colorful portfolio website showcasing 25+ years of enterprise cloud transformation leadership.

## Features

- **Modern Design**: Bold, colorful theme with gradient accents
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile
- **Interactive Elements**: Smooth scrolling navigation and animated sections
- **Contact Form**: Integrated with Formspree for email submissions
- **Sections**: Hero, About, Experience, Education, Certifications, Skills, Recognition, and Projects

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)

## Setup

1. Clone this repository
2. Open `index.html` in your web browser
3. Update the Formspree form ID in `index.html` (line 252) to enable contact form submissions
